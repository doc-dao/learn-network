    重要接口

    netif_rx

    此函数属于驱动与内核的旧接口(非NAPI),主要任务:

    • 对 sk_buff 的数据结构做一些初始化
    • 把已经接受的帧存储到 CPU 的私有输入队列，然后触发相关联的软IRQ NET_RX_SOFTIRQ 以通知内核。
    • 更新相关拥塞登机等统计数据

    net_rx_action

    与 NET_RX_SOFTIRQ相关联。浏览列表中处于轮询状态的设备，然后为每个设备调用相关联的poll函数，以处理入口队列中的帧。每次 poll方法启用时，能处理的帧数目都有最大值。如果在其时间片内无法清空队列，就得等下一次。

    注:

    • 此函数属于 NAPI 相关.
    • NAPI 驱动程序必须提供一个 poll 方法
